# asterisk-configen
Asteriskのconfigを生成して楽に設定できるようなツール

内線番号の増設/減設、接続先SIPトランクの増設/減設、接続先SIPトランクへの発信ルートを「0+3桁」の形式でそれぞれ割り当てる機能を簡単に、GUIでポチポチするだけでAsteriskの設定ファイルに書き込み出来るツール。

docker compose up -dで起動できる。
http://<このツールのサーバIP>:3000へアクセスするとツールのWebUIへアクセス可能。
このツールをインストールするサーバには、Asteriskがインストールされていることが前提です。
生成したコンフィグの書き込みは、/etc/asterisk配下の各コンフィグへsudo権限を用いて、一般ユーザにて書き込みを行う。
